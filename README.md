# PostgreSQL Real-Time Log Analysis System

## Overview
A database-driven application log monitoring system built using PostgreSQL.

## Features
- Normalized relational schema
- 200+ simulated log records
- Index optimization
- Table partitioning
- Stored procedures
- Materialized view
- Advanced analytical SQL queries

## Analytical Capabilities
- Failed login detection
- Peak traffic hour
- Error rate calculation
- Slow API detection
- Top active users per application
- Application-wise error percentage

## Tech Stack
- PostgreSQL 14+
- SQL
